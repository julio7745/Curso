    • _blank: (mesma guia)
      whatsapp

  • Link com dowload

    Para dowload, configuramos os parametros dowload="caminho do arquivo" e type="tipo de arquivo".
    O type serve para dizer o tipo de arquivo ao navegador, visto que, alguns só irão
    fazer o dowload se o tipo de arquivo for especificado e o arquivo realmente for
    do tipo mencionado, isso ocorre por segurança.
    A lista de tipo pode ser pesquisa na internet.
